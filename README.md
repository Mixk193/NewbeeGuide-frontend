# NewbeeGuide-frontend

在学习本系列指南前，请确保你至少掌握了基本的信息搜索能力，本系列指南不会也不可能提供无微不至的关怀，在学习过程中遇到问题请善用搜索。

学习方法：按照章节顺序学习即可。如果章节内使用有序列表列出学习知识点，则推荐按照顺序学习；若使用无序列表，则建议有选择地学习。
